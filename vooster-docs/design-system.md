# 🎨 Smart Road Design System V2.0

> **한 줄 요약:** 신뢰할 수 있는 커리어 멘토 페르소나를 반영한 KOREATECH 학생들을 위한 지능형 학습·경력 관리 플랫폼의 일관된 디자인 시스템

- **버전:** V2.0
- **작성일:** 2025년 1월 1일
- **최종 수정일:** 2025년 1월 1일

---

## 1. 브랜딩 및 아이덴티티

### 1.1. 브랜드 페르소나

**신뢰할 수 있는 커리어 멘토**

데이터를 기반으로 냉철한 분석을 제공하지만, 항상 사용자의 성장을 지지하는 따뜻함과 전문성을 동시에 갖춘 페르소나.

### 1.2. 디자인 키워드

| 디자인 키워드 | 선정 이유 | 시각적/경험적 표현 예시 |
| :--- | :--- | :--- |
| **Reliable (신뢰성 있는)** | 사용자가 자신의 민감한 학업/경력 데이터를 믿고 맡길 수 있어야 하므로, 안정감과 신뢰가 가장 중요하다. | 차분한 Deep Blue 컬러, 예측 가능한 인터랙션, 명확한 정보 구조. |
| **Structured (체계적인)** | 분산된 정보를 논리적으로 통합하고 정리해주는 프로덕트의 핵심 기능을 반영한다. 사용자에게 전문성과 효율성을 전달한다. | 명확한 그리드 시스템, 데이터 시각화(차트, 그래프), 논리적인 정보 위계 설계. |
| **Insightful (통찰력 있는)** | 단순한 데이터 나열을 넘어 AI 분석을 통해 사용자에게 의미 있는 '발견'과 '통찰'을 제공하는 가치를 시각적으로 드러낸다. | 핵심 데이터나 추천 사항을 강조하는 시각적 장치(Accent color, 카드 하이라이트), 직관적인 데이터 시각화. |
| **Empowering (성장을 지원하는)**| 사용자가 자신의 경력을 주도적으로 설계하고 발전시켜 나갈 수 있도록 자신감과 동기를 부여하는 경험을 제공한다. | 명확한 Call-to-Action(CTA) 버튼, 목표 달성률을 보여주는 프로그레스 바, 긍정적이고 격려하는 톤의 마이크로카피. |
| **Clarity (명료한)** | 복잡할 수 있는 AI 분석 결과와 데이터를 모든 사용자가 쉽고 빠르게 이해할 수 있도록 군더더기 없이 명확하게 전달한다. | 충분한 여백 활용, 높은 가독성의 서체, 이해하기 쉬운 아이콘 시스템, 복잡한 용어에 대한 도움말(Tooltip) 제공. |

### 1.3. 서비스 정보

- **서비스명**: 스마트 로드 (Smart Road)
- **타겟**: 한국기술교육대학교(KOREATECH) 재학생

## 6. 아이콘 시스템 (Iconography)

### 6.1. Lucide Icons 기반

- **라이브러리**: Lucide Icons
- **스타일**: Line-based (모서리가 부드러운 선 기반)
- **Stroke Width**: 1.5px
- **기본 크기**: 16px, 20px, 24px

### 6.2. 아이콘 사용 가이드

```tsx
// 16px 아이콘 (sm)
<Home size={16} strokeWidth={1.5} />

// 20px 아이콘 (md)
<User size={20} strokeWidth={1.5} />

// 24px 아이콘 (lg)
<Settings size={24} strokeWidth={1.5} />
```

## 8. 다크 모드 지원

### 8.1. 자동 테마 전환

CSS 변수를 통해 라이트/다크 테마가 자동으로 전환됩니다.

```css
:root {
  --primary: 220 100% 40%;  /* Light theme */
}

.dark {
  --primary: 220 100% 50%;  /* Dark theme - 더 밝은 색상 */
}
```

### 8.2. 다크 모드 사용법

```tsx
// 자동으로 다크 모드 지원
<div className="bg-background text-foreground">
  <h1 className="text-primary">제목</h1>
  <p className="text-muted-foreground">본문</p>
</div>
```

## 10. 접근성 (Accessibility)

### 10.1. 색상 대비

- **일반 텍스트**: 최소 4.5:1 대비율
- **큰 텍스트**: 최소 3:1 대비율
- **UI 요소**: 최소 3:1 대비율

### 10.2. 키보드 네비게이션

- 모든 인터랙티브 요소는 키보드로 접근 가능
- 포커스 표시가 명확하게 보임
- 논리적인 탭 순서

### 10.3. 스크린 리더 지원

- 의미있는 HTML 시맨틱 태그 사용
- 적절한 ARIA 라벨 제공
- alt 텍스트 제공
